## Installation *Need internet connection
- Open your terminal
- Run "composer install" 
- Run "php artisan key:generate" 
- Set db config in .env
- Run "php artisan migrate"
- Run "php artisan db:seed"
- Run "npm install"
- Run "npm run dev"
